#### 記住!!一律都看第一個判斷真值和假值!!!!!!
### && (and) 
```
若第一個不是真值，就回傳第一個
若第一個是真值，就回傳第二個
ex: 0 && 1 // 0
1 && 2 // 1
```

### || (or)
```
若第一個不是真值，就回傳第二個
若第一個是真值，就回傳第一個
```

### !  (not)
```
顛倒否定的概念
!0 // true
!1 // false
```

## 預設值為0的解決方法
使用三元運算子<br/>
```
var originCash = 500;
function updateEasyCard(cash) {
  cash = parseInt(cash);
  console.log(cash);
  
  // 當 cash 是數值或為 0 時，使用 cash 的數值
  // 如果 cash 是 NaN 時，則直接套用 500
  cash = (cash || cash === 0)? cash: 500;
 
  var money = cash + originCash;
  console.log('我有 ' + money + ' 元');
}
updateEasyCard(0);
```
