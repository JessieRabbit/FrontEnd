# 設計多欄式版型
https://codepen.io/jessierabbit/pen/VoPyQJ<br/>
子元素設定float會看父元素的寬度，若不夠寬會推到下面<br/>
希望元素和元素之間想要並排，可以用float的應用時機<br/>
```
.floatbox1 {
    width: 200px;
    background: black;
    height: 50px;
    float: left;
}

.floatbox2 {
    width: 200px;
    background: pink;
    height: 50px;
    float: left;
}
```
